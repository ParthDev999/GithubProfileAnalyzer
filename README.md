# 🚀 GitHub Developer Analytics Dashboard

An advanced GitHub analytics platform built using **HTML, CSS, and JavaScript** that provides deep insights into GitHub profiles, repositories, languages, achievements, and developer activity.

🔗 Live Repository: https://github.com/ParthDev999/GihubProfileAnalyzer

---

## 📌 Overview

GitHub Developer Analytics Dashboard allows users to search any GitHub profile and instantly view detailed analytics, repository statistics, language usage, achievements, activity scores, and profile comparisons through a modern and responsive user interface.

The project leverages the GitHub REST API to fetch real-time profile and repository data and presents it in an intuitive dashboard format.

---

## ✨ Features

### 👤 Profile Analytics
- GitHub profile information
- Profile completeness score
- Developer rank system
- Account creation details

### 📊 Repository Insights
- Repository statistics
- Stars, forks, issues, and repository size
- Favorite repository detection
- Top repositories showcase

### 💻 Language Analytics
- Most-used language detection
- Language distribution visualization
- Developer role prediction based on repository activity

### 🏆 Achievement System
- Dynamic achievement badges
- Activity-based rewards
- Open-source contribution indicators

### 📈 Developer Metrics
- Activity Score calculation
- GitHub Scorecard
- Community impact analysis
- Repository growth insights

### 🔍 Advanced Search & Filtering
- Repository search
- Language-based filtering
- Sorting by:
  - Stars
  - Forks
  - Issues
  - Repository size
  - Recently updated

### ⚔️ User Comparison
- Compare two GitHub users side-by-side
- Followers comparison
- Repository comparison
- Stars and forks comparison
- Activity score comparison

### 🎨 Modern UI/UX
- Glassmorphism design
- Floating animated background elements
- Responsive design
- Dark/Light mode
- Animated statistics counters
- Search history support

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/ParthDev999/GihubProfileAnalyzer.git
```

Navigate to the project folder:

```bash
cd GihubProfileAnalyzer
```

Open:

```bash
index.html
```

in your browser.
